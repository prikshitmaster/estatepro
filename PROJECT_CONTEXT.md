# EstatePro CRM

## What This Is
A CRM for real estate brokers and agencies.

## Main Goal
Help brokers manage leads, properties, deals, and follow-ups in one system.

## Current Phase
MVP Build

## Target Users
- Small brokers
- Agencies
- Solo agents

## Core Promise
Simple, fast, mobile-friendly CRM.

## Tech Stack
- Next.js
- Tailwind CSS
- Supabase
- Vercel
